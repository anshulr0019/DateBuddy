## 📱 What is Dil Se?

**Dil Se** is a complete, production-ready mobile dating application designed specifically for the Indian Gen Z market (ages 18-30). Built with Next.js 16, TypeScript, PostgreSQL, and Tailwind CSS.

### Key Features
- ✅ **38+ Screens** - Complete UX flow
- ✅ **4 Major User Flows** - Onboarding, Discovery, Chat, Profile
- ✅ **18 Database Tables** - Full backend schema
- ✅ **Premium Subscription** - ₹50/day pricing
- ✅ **Mobile-Optimized** - 390×844 iPhone standard
- ✅ **Gen Z Design** - Psychedelic pastels, emoji-rich
